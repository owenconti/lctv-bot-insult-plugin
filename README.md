# lctv-bot-insult-plugin
Insult plugin for the LCTV Bot

## Setup
Add a `settings.json` file to the plugin's directory with this structure:

```
{
    "useSayCommand" : false,
    "insultVoice" : "Daniel",
    "insults" : [
    	"You stink!",
    	"You're so ugly you look like a regex!"
    ]
}
```
